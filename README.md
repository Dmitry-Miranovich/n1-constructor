# Конструктор N1

Проект создан с помощью [Create React App](https://github.com/facebook/create-react-app).

## Доступные скрипты

### `npm start`

Запускает приложение в режиме разработки.\
Открыть [http://localhost:3000](http://localhost:3000) для просмотра в браузере.

### `npm run build`

Собирает приложение для продакшена в папку `build`.

### `npm test`

Запускает тестовый runner в режиме watch.

### `npm run eject`

**Односторонняя операция!** Удаляет зависимость от Create React App.

### `npm run server`

Запускает JSON-server на порту 3001 для работы с mock API.

### `npm run dev:full`

Запускает одновременно React приложение и JSON-server.

## Быстрый старт

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npm start
```

Приложение откроется на [http://localhost:3000](http://localhost:3000), а API сервер на [http://localhost:3001](http://localhost:3001).

## Примечание

Для перехода на страницу админа в URL строке пропишите путь {текущий URL}/admin

В связи с требованиям задания и ограничением mockAPI - было предусмотренно отключить создание новых блоков для главной страницы, в связи с дублированием данных и отсутствием функциональной базы данных для создания связей по внешним ключам

## Технологии

- React 18
- React Router DOM
- Zustand (управление состоянием)
- JSON-server (mock API)
- SCSS (стилизация)
